在这里看到这么一个计算UTF-8字符串长度的程序：

[c]
int my_strlen_utf8_c(char *s) {
int i = 0, j = 0;
while (s[i]) {
if ((s[i] & 0xc0) != 0x80) j++;
i++;
}
return j;
}
[/c]

不解。查wikipedia，得一表：
